# Event Management System

## Project Description
This is a simple Event Management System web project made using HTML, CSS, JavaScript, Node.js and Express.js.  
It allows users to add events with name, date and time and display them in a list format.

The project also includes a backend server using Express.js to serve the frontend files.

---

## Features
- Add event name, date and time  
- Display added events in list  
- Delete event from list  
- Simple and clean user interface  
- Backend server using Express.js  

---
